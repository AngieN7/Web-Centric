  • jQuery

    What is it: jQuery is a fast, small and feature-rich JavaScript library. Event Handling, animation and HTML document traversal are much simpler with an easy-to-use API that works across a variety of browsers.

    Purpose: Simplify client-side scripting of HTML. By abstracting complex JavaScript tasks into methods, it can be called with less code and fewer complexity.

    Use Case: Used for tasks such as selecting DOM elements, creating animations, handling events, and developing Ajax applications.

    jQuery useage has declined with the rise of modern frameworks like React, Angular, and Vue.js



    In this project: A jQuery plug-in was used for validating the contact information form and AJAX sends it as an email.
